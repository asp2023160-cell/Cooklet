Cooklet - Digital Recipe Book
Cooklet is a modern, interactive web application designed to preserve and share authentic recipes, with a special focus on Sri Lankan culinary heritage. It provides a clean, user-friendly interface for browsing categories, searching for specific dishes, and managing a personal digital cookbook.

🚀 Features
Interactive Recipe Gallery: Browse through a collection of recipes with high-quality imagery and detailed instructions.

Dynamic Filtering: Easily filter recipes by categories such as Breakfast, Lunch, Dinner, and Dessert.

Real-time Search: Find recipes or specific ingredients instantly using the integrated search bar.

Add New Recipes: A built-in modal form allows users to contribute their own recipes, including image URLs, ingredient lists, and step-by-step instructions.

Responsive Design: Fully optimized for all devices, from desktops to mobile phones, using Bootstrap 5.

Hero Carousel: Showcases featured recipes like Traditional Kiribath and Ceylon Crab Curry on the home page.

🛠️ Technology Stack
Frontend: HTML5, CSS3 (Custom Styles)

Framework: Bootstrap 5

Icons: Bootstrap Icons

Typography: Google Fonts (Poppins & Playfair Display)

Scripting: JavaScript (Vanilla)

📖 How to Use
Browse: Use the index.html page to explore featured dishes or jump straight into a category.

Search: Navigate to the Recipes page to search for specific ingredients or dish names.

Contribute: Click the "Add Recipe" button to open the submission form. Fill in the details and save it to your local session.

View: Click on any recipe card to open a detailed view containing the full ingredient list and preparation steps.
